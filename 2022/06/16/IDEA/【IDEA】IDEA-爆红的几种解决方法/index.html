IDEA 爆红的几种解决方法

检查 pom.xml的图标是否是蓝色的小m，如果不是则说明未以Maven项目加载

解决方法：找到 pom.xml -> 右键-> Maven-> addAsMavenProject

检查项目所用的 JDK 版本是否正确

解决方法：project structure -> Project

检查 Maven 的位置（以及选择的版本）

解决方法：Preferences -> Build, Execution, Deployment -> Build Tools -> Maven
