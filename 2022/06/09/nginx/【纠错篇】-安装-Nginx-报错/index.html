【纠错篇】 安装 Nginx 报错

本文最后更新于：a month ago

在 CentOS8 上安装 Nginx 遇到的问题以及解决办法

  • 注意：这边可能会遇到两个问题，如如果遇到请参考下面方法，没有遇到就跳过：

对于问题 1：进入到:/解压缩后的文件夹下的/objs下，使用 vim 打开 Makefile 并删除下列代码中的-Werror

# 删除前
CFLAGS =  -pipe  -O -W -Wall -Wpointer-arith -Wno-unused-parameter -Werror -g　
# 删除后
CFLAGS =  -pipe  -O -W -Wall -Wpointer-arith -Wno-unused-parameter -g　

问题 2：进入到下方文件夹，使用 vim 打开ngx-user.c后注释掉下面一行代码

cd.current_salt[0] = ~salt[0];

两个问题修改之后的效果：

问题已解决！
