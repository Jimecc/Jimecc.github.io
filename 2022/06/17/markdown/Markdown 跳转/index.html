MarkDown 写作笔记（链接篇）



普通外链

[超链接名字](连接 url)

示例：

[点我打开百度](www.baidu.com)

点我打开百度

注意事项：记得加上https://或者http://





文章内跳转

方式一（不推荐）

使用 <a>标签直接跳转到某一个目录：

示例：

<a href="#文章内跳转">点我跳转到【文章内跳转】</a>

点我跳转到【文章内跳转】

这种方式仅在 Markdown 文件内有效，放在博客中转换成 html后可能会失效。

方式二

对目录设置锚点

<h3 id="123">这是锚点</h3>
<a href="123">用方式一跳转</a>
[用方式二跳转](#123)

这是锚点

用方式一跳转
用方式二跳转
