【维护小破站】博客重新搭建

本文最后更新于：a month ago

前言 吐槽一下遇到的离谱问题

很离谱！很奇葩！把本地的博客放到了TencentCloud之后，默认语言变成了阿拉伯文！

找了很久的问题，最后确认是NexT主题配置文件出了问题，最后索性直接重新搭建了。



前期准备相关配置及版本

相关配置：

  • MacBook M1 Air
  • NexT version 8.8.0
  • Tencent Cloud CentOS 8.2 64bit
  • Nodejs 14.16.1
  • 文件传输：宝塔


开始安装

安装 宝塔

在TencentCloud 安装Hexo与NexT主题很简单，使用git与vim便可以完成，但是仅仅使用vim来修改配置则太麻烦了，况且美化的过程工作量还不小，询问过【狐狸】之后，采取了他的办法，用宝塔来实现一个简单的“桌面”，宝塔的安装命令如下：

# CentOS
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
# Ubuntu/Deepin
wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
# Debian
wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh
# Fedora
wget -O install.sh http://download.bt.cn/install/install_6.0.sh && bash install.sh

安装Hexo

  • 安装Nodejs
  • 使用npm直接安装hexo
  • 详情请见【Hexo博客安装】

安装NexT主题

  • 去NexT主题仓库克隆下来

git clone https://github.com/next-theme/hexo-theme-next.git

  • 打开Hexo配置文件 并将theme改为next

![图片](/Users/hsk/Library/Application Support/typora-user-images/MyMac_2021-10-07 下午4.44.57.png)

  • 更多配置详情请看【NexT主题升级】
