note

效果：

default

primary

success

info

warning

danger

_config 文件配置关键字：note, 我的配置如下：

style: flat
icons: true
border_radius: 3

代码：

{% note default %}
default
{% endnote %}
{% note primary %}
primary
{% endnote %}
{% note success %}
success
{% endnote %}
{% note info %}
info
{% endnote %}
{% note warning %}
warning
{% endnote %}
{% note danger %}
danger
{% endnote %}

label

_config 文件配置关键字：Label , 需要用的话把值设为true即可。
效果：

默认 主要 成功 信息 警告 危险

代码：

{% label default@默认 %}
{% label primary@主要 %}
{% label success@成功 %}
{% label info@信息 %}
{% label warning@警告 %}
{% label danger@危险 %}
